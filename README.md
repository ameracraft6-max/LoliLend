# LoliLend

Gaming portal desktop application for Windows.

## Features

- System monitoring (CPU, RAM, GPU, FPS, temperature, network)
- In-game overlays (FPS, crosshair, ping, network speed, temperature)
- Game analytics and session tracking
- Discord Quest tracker
- Yandex Music Discord Rich Presence
- Telegram SOCKS5 proxy
- AI chat (Cloudflare Workers AI)
- Windows autostart, task scheduler, hosts manager
- Clipboard manager
- Profile system with 4 visual themes
- License system with HWID binding
